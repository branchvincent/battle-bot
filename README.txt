## Command Interface ##

BRSTbot comes with a command interface, allowing you to enter commands through the Serial Moniter

| Command | Description |
| ------- | ----------- |
| speed37 | Sets the speed to 37. Accepts any number from 0 to 255. |
| dirforward | Sets the motor direction. Accepts (forward, back, stop) |
